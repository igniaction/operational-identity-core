# Interoperability

---

# Overview

Operational Identity Core was intentionally designed to
support interoperability across heterogeneous environments.

---

# Supported Integration Patterns

- HTTP APIs
- JWT payload propagation
- CLI orchestration
- containerized runtimes

---

# Future Integrations

- OpenID Connect
- OAuth2
- SAML
- API gateways
- service mesh architectures

---

# Architectural Goal

The operational capability should remain transportable
between runtimes and infrastructure providers.
